General Customization

Property Possible Values Description
box-sizing border-box Can be applied at the start of the website to all elements (*). Makes size of an element more predicable by resizing the content to have a whole number and not to be modified/added by margin-padding
font-style Can be all the possible fonts that exist. Can use a font from google and apply a fallback if the font can't be rendered like sans or sans-serif.
font-weight Can be a number or 'bold'. Details the thickness of a word.
margin Four possible values that detail the margin top-right-bottom-left. Detail the space between the selector and the other elements.
padding Four possible values that detail the padding top-right-bottom-left. Detail the space between the content contained on the selector and the border of the selector.
background-image 'url(url...)' Details a background-image that will be contained on the selector.
max-width or min-width number, percentage, rem, em The website adapts when the page increase or decreases. This property stops the content from adapting when reaching a max/min width..
border-radius number values Details how rounded the corners of a border will be. 50px makes a circle.
display 'inline-block', 'block' Details if certain elements can be together in the same line or will span the whole width of the page.
list-style 'none' Removes the default option for bullet points/numbers for the ordered/unordered lists.
list-style-position 'inside' This applies to ul or ol. It details that the bullet points should be inside the parent container of the ul/ol.
opacity 0-1 1 is full opaque, full resolution. 0 value is full transparent, can't see it.
text-transform 'uppercase' - 'lowercase' transforms the text to all uppercase or lower case.
text-align 'center' - 'right' - 'left' Align the content based on the parent element of the selector.
text-transform 'uppercase' - 'lowercase' transforms the text to all uppercase or lower case.
letter-spacing -- word-spacing -- line-height Number Modifies the space between the words. The number is a ratio. If want to increase the letter spacing of double the height of each character, then the value will be 2 and if the value is 1 then everything stays the same.
border widthBorder - styleLine(solid/dashed) - color Need to set table, th, and td to have a border to show the gridlines in the browser.

Table Customization

Property Possible Values Description
border-collapse 'collapse'/'separate' If set to 'collapse' then cells inside a table will share borders. If set to 'separate', then each cell will have different borders. This only applies when targerting the table directly.
border-spacing numer If this is 0, it will appear as if there is no separation between cells.
